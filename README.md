# Imperium Sui

Стартовый Flutter-проект, созданный в этой директории и готовый к дальнейшей
разработке.

## Что уже настроено

- создан Flutter app-проект с именем пакета `imperium_sui`
- инициализирован git-репозиторий
- оставлен чистый стартовый экран вместо demo-counter шаблона
- добавлены базовые editor-конфиги для удобного старта

## Быстрый старт

```bash
flutter pub get
flutter run
```

Если нужно явно выбрать платформу:

```bash
flutter run -d windows
flutter run -d chrome
```

## Полезные команды

```bash
flutter analyze
flutter test
flutter pub outdated
```

## Основные файлы

- `lib/main.dart` - точка входа и временный стартовый экран
- `test/widget_test.dart` - базовый smoke test
- `pubspec.yaml` - зависимости и метаданные проекта
